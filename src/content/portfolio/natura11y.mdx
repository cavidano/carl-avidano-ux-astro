## Why I built Natura11y

Natura11y is an accessibility-first design ecosystem I designed and built to keep design decisions, component behavior, and production code aligned. As my passion for accessibility grew, so did my desire to become a stronger front-end developer and understand how design systems work from conception through production.

What began as a personal learning project has grown into an open-source ecosystem containing shared CSS and JavaScript foundations, Figma libraries, an icon system, React components, Astro documentation, Storybook, and coordinated package releases.

The project represents the full design-system lifecycle: defining principles, creating design and code foundations, engineering accessible interactions, documenting component behavior, distributing packages, and applying the system to production websites.

## An end-to-end design system

Over time, Natura11y has grown into an end-to-end design system. It's been used in real world project such as Cheetah Conservation Fund, Visionlearning, and others. 

Naturally supports every part of the project life cycle: from low-fidelity ideation and design exploration to production-ready code.
